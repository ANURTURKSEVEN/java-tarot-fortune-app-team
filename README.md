# java-tarot-fortune-app-team

Java Swing tabanlı, takım çalışması olarak geliştirilmiş bir fal uygulamasıdır. Projede tarot ve  kahvefalı, rüya ve burç yorumu ile ilgili modüller yer almaktadır.

## Özellikler
- Tarot kartı seçme ve yorumlama
- Kahve falı arayüzü
- Rüya metnini kelime analizi ile yorumlama
- Burç hesaplama
- Swing tabanlı masaüstü arayüz
- Nesne yönelimli programlama yapısı

## Kullanılan Teknolojiler
- Java
- Swing
- OOP
- File I/O

## Projedeki Katkım
- Tarot kartı seçme kısmını geliştirdim.
- Fal seçim ve yorumlama ekranlarını tasarladım.
- Tarot yorumlama mantığının oluşturulmasına katkı sağladım.
- Uygulamanın kullanıcı arayüzü ve akışında görev aldım.

## Çalıştırma
1. Projeyi bilgisayarına indir.
2. `src` klasörünü IDE ile aç.
3. `TarotApp.java` veya ana sınıfı çalıştır.

## Not
Bu proje bir takım çalışmasıdır. Repo, geliştirme sürecinde kullanılan kaynak kodları içermektedir.
